# Projekt - Grafika Komputerowa

## Odplanie

```bash
cd build && make && ./grafika_app
```


## Biblioteki 
```bash
brew install glfw glew glm
```

## Kompilacja

1. Utwórz folder budowania:
   ```bash
   mkdir build && cd build
   ```
2. Wygeneruj pliki Makefile za pomocą CMake:
   ```bash
   cmake ..
   ```
3. Zbuduj projekt:
   ```bash
   make
   ```
4. Uruchom aplikację:
   ```bash
   ./grafika_app
   ```
